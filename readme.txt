A simple plugin that adds a click to call bar for mobile visitors. 

== Description ==
This plugin adds a customizable click to call bar to the bottom of the window for mobile users. It will allow users to easily call you from their phone. It automatically adds events if you\'re using Google Analytics universal. 

== Installation ==
1. Navigate to your WordPress dashboard and go to Plugins > Add New > and search for Really Simple Click to Call. 
2. Activate the plugin through the \'Plugins\' screen in WordPress
3. Use the Settings->Really Simple Click to Call screen to configure the plugin
4. Click enable and enter your phone number, message and choose your color options.
5. Click Save Changes and you\'re done!

== Frequently Asked Questions ==
Where do I enter my phone number and message? 

Enter your phone number and message in the Settings > Really Simple Click to Call page. If you don\'t enter a message the phone icon will still show up. 

How should I enter my phone number?

Simply enter the numbers like this: 1234577891

Can I remove the phone icon?

Not in the plugin settings page however you can remove it with custom CSS. 

What font will it use?

It will the inherit the font from your theme. You can change this with custom CSS in your theme.

Do I have to do anything to setup the event tracking in Google Analytics?

No, it\'s automatically set up to work with Universal Analytics. Once your plugin is setup and configured, you will be able to see all the clicks in the Google Analytics events section.
